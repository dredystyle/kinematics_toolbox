deg2quat

Convert axis-angles or plain degree into the corresponding quaternion values. Can be used with a plain number or with an axis angle.

Contents

  • Syntax
  • Description
  • Input Arguments
  • Output Arguments
  • Notes
  • Examples

Syntax

quaternion = deg2quat(degree)

Description

Convert axis-angles or plain degree into the corresponding quaternion values.

Input Arguments

inDeg -- Input angle in degrees.

Can be float, vector, or matrix

Output Arguments

outQuat -- Corresponding quaternion values

Notes

More info under http://en.wikipedia.org/wiki/Quaternion

Examples

deg2quat([10, 20, 30])

ans =
    0.0872    0.1736    0.2588
