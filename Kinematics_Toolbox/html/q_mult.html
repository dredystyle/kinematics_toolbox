q_mult

Quaternion multiplication If one or both of the quaternions have only three columns, the scalar component is calculated such that the lenght of the quaternion is one.

Contents

  • Syntax
  • Input Arguments
  • Output Arguments
  • Notes

Syntax

  p=q_mult(q,r)

Input Arguments

  • q -- First input quaterion
  • r -- Second input quaterion

Output Arguments

  • p -- Quaternion product q*r

Notes

If either of the inputs contains only the quaternion vector, the input is expanded to unit quaternions.
